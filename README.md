# Dating-application
First and foremost make sure you have django install into your command bash. (pip install django) 
to run the application activate the virtual environment .Venv\Scripts\Activate.bat 
set the directory to cd Myproject 
make sure templates images css and js are listed under static folder.
there should only be one folder for templates on text editor just move the login.html to the folder with the other html templates.
views,models,admin, signals forms, urls.py should  all be under myapp folder . 
for this github I have made sure that the file directory is easy and attainable. 
if the app is not running make sure you to do  python manage.py makemigr
ations so the backend can update with the templates front-end. after do python manage.py and runserver.
